# 職務経歴 / Resume

伊藤 達章の職務経歴書サイトです。

## URL

https://gratachu.github.io/resume/

## 技術スタック

- HTML/CSS (バニラ)
- GitHub Pages
- GitHub Actions (自動デプロイ)

## デプロイ

`main` ブランチへのプッシュで自動的にGitHub Pagesにデプロイされます。

## ローカルでの確認

```bash
# シンプルなHTTPサーバーを起動
python3 -m http.server 8000
# または
npx serve .
```

ブラウザで `http://localhost:8000` を開いて確認できます。

## 更新方法

1. `index.html` を編集
2. コミット＆プッシュ
3. GitHub Actionsが自動的にデプロイ

## ライセンス

個人の履歴書サイトです。
